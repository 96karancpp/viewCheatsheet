  3. Axios Package installer in Vue:

    First you need to install it in your package through command line and thenafter you have to include it in components where you want to fire an API

    npm install --save axios vue-axios
import Vue from 'vue'
import axios from 'axios'
import VueAxios from 'vue-axios'
Vue.use(VueAxios, axios)
  4. JSON SERVER in VueJS:

    First, Install Json server and then Start JSON Server and use your database name as it uses db.json

    npm install -g json-server 
json-server --watch db.json
  5. Vue Router:
